Variable in Python

Variables in Python are used to store data values,such as numbers,strings,or lists,which can be referenced and manipulated in a program.

There are two types of Variables in Python.

  • Local Variable

    • In Python, a global variable is a variable that is defined outside of a function and can be accessed and modified from any function within the same script or module.
    • Modifying a Global Variable Inside a Function By default, if you try to modify a global variable inside a function, Python treats it as a local variable, which can lead to errors. To modify a global variable inside a function, you must use the global keyword.
  • Global Variable
    • A local variable is a variable that is declared inside a function and is only accessible within that function. It cannot be used outside the function in which it is defined.
    • By default, if you try to modify a global variable inside a function, Python treats it as a local variable, which can lead to errors. To modify a global variable inside a function, you must use the global keyword.
